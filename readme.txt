++++FOR MAC OS++++
- Install Perl
    - Command:        curl -L http://xrl.us/installperlosx | bash
    - Reference:
        + https://learn.perl.org/installing/osx.html
        + http://macappstore.org/perl/


- Install perl modules
    - Basic steps:
        1. Open a terminal(Ctrl-Alt-t).
        2. Enter the command cpan.
        3. At the prompt cpan[1]>, type install Switch.
        4. Once completed, Type exit.
    
    - Modules:
        - "Switch" 
        - "File" 
        - "Time"
